## Clone and review it

Run the following commands to clone and run it.

### To clone the repository

`git clone git@github.com:adamavip/house_app.git`

### To enter Directory

`cd house_app`

### To Install Packages

`npm install`

### To run project on local server

`npm start`

### To run project tests

`npm test`

### To open project with vs code

`code .`
